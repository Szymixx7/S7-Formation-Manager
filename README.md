# S7-Formation-Manager
S7 Formation Manager to aplikacja do tworzenia i zarządzania składami piłkarskimi wielu drużyn. Umożliwia dodawanie zawodników, kontuzji i zawieszeń, ustawianie formacji na interaktywnym boisku, budowanie składów ogólnych i meczowych oraz automatyczny zapis danych każdej drużyny w osobnym folderze.
